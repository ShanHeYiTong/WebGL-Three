环境搭建：
1.安装 Three.js
      npm i three --save
      npm i @types/three --save-dev
      
2.安装 OrbitControl
      npm i three-orbitcontrols-ts --save 
      
3.组件中引用
      import * as THREE from 'three';
      import { OrbitControls } from 'three-orbitcontrols-ts';
